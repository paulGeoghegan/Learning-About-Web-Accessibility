Learning about accessibility with images

The image below has no alt text so someone using a screen reader wouldn't be able to tell what it is:

This is equivilent to someone putting a blank image on a website and expecting you to know what it is:

Thats why it's important in order to add alt text you just have to have something like this inside your image tag:

src="example.jpg" alt="This is an image"

Now we can add some alt text to the original image

Now when someone with a screen reader navigates to the image they would hear: A fluffy grey and white kitten sitting on a tree stump looking up while surrounded by trees

It's that simple so make sure you remember why it's important
