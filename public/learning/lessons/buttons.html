Making buttons accessible

Almost every website you will come across will have buttons but allot of the time they aren't accessible. This is especially tru when the website is using different icons for the buttons but why is this?

It's simple when someone adds an icon to a button they don't realise that either the text in the icon is now part of an image which the screen reader can't read or that a screen reader can't tell the user what the image is.

There are 2 main ways to fix this:

  1. Using Value

    This can be used to specify the text on a button like:

    value="Click Me!"
  2. Using aria-label

    This will add some text to the button that only the screen reader will read and is usefull when you have an icon and don't want to add any text to the button

    aria-label="Click Me!"
