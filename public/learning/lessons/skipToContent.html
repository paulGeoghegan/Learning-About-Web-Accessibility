The secret skip to content link

Something you might not know if you have never used a screen reader is that some websites have a hidden link at the top of the page which is there just for screen reader users

This is extremely useful as on a lot of websites they have huge nav bars and a lot of not so useful links and information at the top before you actually get to the content of the page so a screen reader user has to navigate past all of this which can be annoying and time consuming. The fix? The skip to content link.

It works by having at least in the case of my websites a div with an id of middleOfPage or something like this. This div is where you will put all of your content for that particular page. After you have that just add a link to the top of your page

Skip To Top

This link takes you back to the top of the page since we are already in the middleOfPage div. All the code you need for this is a link with the following:

href="#idOfElement"

Changing the idOfElement to whatever the id of your element or div is.

Another important thing is to make this invisable so that everyone else doesn't have to look at it. The easiest way to do this is the following css:

#skipToContent{font-size: 0;}

This will require that you give your element the same id as you reference in your css but once you have done that it should be working.
